# BroBot ToDo — план доработки

## Фичи

### 1. Задачи без даты → сегодня
Если LLM не определила дату — автоматически ставится сегодня.
- Все задачи (не только явные напоминания)
- Due date = сегодня 20:00 UTC (по существующей логике "только дата")

---

### 2. Команда /todotoday — задачи на сегодня
Список задач с дедлайном сегодня + инлайн-кнопки под каждой.
- `✅` — отметить выполненной
- `✏️ Название` — изменить название
- `📅 Дата` — изменить дату
- Редактирование: бот просит написать новое значение, затем обновляет задачу в MS Todo

---

### 3. Пересланные и Reply сообщения
**Reply** (ответ на сообщение в чате с ботом):
- Уже частично работает, будет доработано

**Forward** (пересылка из другого чата):
- Обнаружение по полю `forward_origin`
- Если есть комментарий пользователя → title = комментарий, description = пересланный текст целиком
- Если комментария нет → обрабатывать пересланный текст как обычное сообщение

---

### 4. Улучшение LLM
- Не сокращать сообщения: одно действие (даже развёрнутое) → весь текст в title
- Сокращать title только если несколько независимых действий
- Использовать слова пользователя, не переформулировать
- Новое поле `confidence: "high" | "low"` — уверенность в классификации
- Новое поле `subtasks: string[] | null` — подзадачи из явных списков

---

### 5. Подтверждение перед созданием
Перед созданием задачи бот показывает превью и спрашивает:
```
📋 Создать задачу?

📌 Название
📅 Дата
📝 Описание
☑️ Подзадачи

[✅ Создать]  [✏️ Изменить]  [❌ Отмена]
```
- `✏️ Изменить` → бот просит написать исправленное название → показывает превью заново
- Данные ожидающей задачи хранятся в n8n Data Store

---

### 6. Настройки бота — /settings
Команда `/settings` показывает текущие настройки с кнопками переключения.

**Режим подтверждения:**
- `Все задачи` (по умолчанию) — подтверждение для каждой задачи
- `Только неуверенные` — только когда LLM не уверена (confidence = low)
- `Отключить` — создавать сразу без подтверждения

Настройки хранятся в n8n Data Store (ключ `settings_{chat_id}`).

---

### 7. Статистика — /stats
```
📊 Статистика

✅ Выполнено сегодня: 3
🆕 Создано сегодня: 5
📋 Открытых задач: 12
⚠️ Просрочено: 2
```

---

### 8. Просроченные задачи — /overdue
Список задач с прошедшим дедлайном, статус не выполнено.
- Показывает дату дедлайна рядом с названием
- Кнопка `✅ Выполнено` под каждой задачей

---

### 9. Подзадачи (checklistItems)
Если в сообщении явный список — создаются подзадачи в MS Todo:
- Нумерованный список: `1. ... 2. ...`
- Буллеты: `- ...` или `• ...`
- Перечисление через запятую после двоеточия: `купить: молоко, хлеб, масло`

Реализация через HTTP Request к MS Graph API (POST checklistItems).
Каждый пункт отмечается в MS Todo отдельно.
